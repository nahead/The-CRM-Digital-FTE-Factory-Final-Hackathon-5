# TechCorp SaaS - Company Profile

## Company Overview
**Name:** TechCorp Solutions Inc.  
**Industry:** B2B SaaS (Project Management Software)  
**Founded:** 2020  
**Employees:** 150  
**Customers:** 5,000+ businesses worldwide

## Product: ProjectFlow
A cloud-based project management and collaboration platform designed for remote teams.

### Key Features:
- **Task Management:** Create, assign, and track tasks with custom workflows
- **Team Collaboration:** Real-time chat, file sharing, and video calls
- **Time Tracking:** Built-in time tracking and reporting
- **Integrations:** Connects with Slack, Google Workspace, Microsoft Teams, GitHub
- **Mobile Apps:** iOS and Android native apps
- **API Access:** RESTful API for custom integrations

## Pricing Tiers
- **Starter:** $10/user/month - Up to 10 users, basic features
- **Professional:** $25/user/month - Unlimited users, advanced features
- **Enterprise:** Custom pricing - Dedicated support, SSO, custom integrations
