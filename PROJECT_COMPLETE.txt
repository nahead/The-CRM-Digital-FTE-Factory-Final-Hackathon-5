## NEXT STEPS TO RUN

### STEP 1: Start Docker Desktop
   - Open Docker Desktop application
   - Wait for it to fully start
   - Verify: docker ps (should work)

### STEP 2: Get Gemini API Key (REQUIRED)
   1. Visit: https://makersuite.google.com/app/apikey
   2. Click "Create API Key"
   3. Copy the key
   4. Edit .env file
   5. Add: GEMINI_API_KEY=your_key_here

### STEP 3: Start Infrastructure
   docker-compose up -d

### STEP 4: Install Python Dependencies

   OPTION A (Recommended for quick start):
   venv\Scripts\activate
   pip install fastapi uvicorn google-generativeai python-dotenv httpx

   OPTION B (Full installation - requires Visual Studio Build Tools):
   venv\Scripts\activate
   pip install -r requirements.txt

### STEP 5: Start API
   cd src
   python -m uvicorn api.main:app --reload --port 8000

### STEP 6: Test
   # In another terminal:
   curl http://localhost:8000/health

   # Or run test suite:
   python test_api.py

## DEPLOYMENT TO KUBERNETES

1. Build Docker image:
   docker build -t customer-success-fte:latest .

2. Push to registry:
   docker tag customer-success-fte:latest your-registry/customer-success-fte:latest
   docker push your-registry/customer-success-fte:latest

3. Update k8s/deployment.yaml with your image

4. Deploy:
   kubectl apply -f k8s/deployment.yaml

5. Verify:
   kubectl get pods -n customer-success-fte
   kubectl get svc -n customer-success-fte
